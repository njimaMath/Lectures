上限・下限と有界性 ─ 概要ノート

1. 最大値と最小値（$\max$, $\min$）

定義: 非空集合 $A \subset \mathbb{R}$ について、

  • $a = \max A$ とは $a \in A$ かつ $\forall x \in A,\; x \le a$。
  • $a = \min A$ とは $a \in A$ かつ $\forall x \in A,\; a \le x$。

注意: $\max A$ と $\min A$ は常に集合の要素であり、存在しないこともある。

例:

  • $\max\{1,2,3\}=3$
  • 開区間 $(0,1)$ に最大値・最小値は存在しない

2. 上限（$\sup$）と下限（$\inf$）

定義: 上に有界な非空集合 $A \subset \mathbb{R}$ について、$s = \sup A$ とは $$ \begin{cases} \forall x \in A,\; x \le s,\\ \forall \varepsilon > 0,\; \exists x \in A \text{ s.t. } s - \varepsilon < x. \end{cases} $$ 下限 $i = \inf A$ は同様に $$ \begin{cases} \forall x \in A,\; i \le x,\\ \forall \varepsilon > 0,\; \exists x \in A \text{ s.t. } x < i + \varepsilon. \end{cases} $$

例:

  • $\sup(0,1)=1$（ただし $1 \notin (0,1)$）
  • $\inf\{1/n : n\in\mathbb{N}\}=0$
  • $\sup\{1-1/n : n\in\mathbb{N}\}=1$

3. 上限・下限に関する証明例

3.1 $\sup\{1/n : n\in\mathbb{N}\}=1$

上限であることの確認: 任意の $n$ で $1/n \le 1$。したがって $1$ は上界。

最小上界性: 任意の $\varepsilon>0$ に対し、$N>1/\varepsilon$ を選ぶと $1/N > 1-\varepsilon$。よって条件を満たす要素が存在。

3.2 $\inf\{1-1/n : n\in\mathbb{N}\}=0$

下界は $0$、$n$ を大きくすると $1-1/n$ は任意に $0$ に近づく。

4. 有界性

定義:

  • 上に有界: $\exists M \in \mathbb{R}$ で $\forall x \in A,\; x \le M$。
  • 下に有界: $\exists m \in \mathbb{R}$ で $\forall x \in A,\; m \le x$。
  • 有界: 上にも下にも有界。

例:

  • 区間 $(0,1)$ は上下とも有界
  • 自然数集合 $\mathbb{N}$ は下に有界だが上に有界でない

5. 連続性公理（完備性）

実数 $\mathbb{R}$ は次を満たす：
「上に有界な非空集合には必ず最小上界が存在する」。

近接定理: 有界集合 $A$ と $\varepsilon>0$ に対し、 $$ \exists b \in A \text{ s.t. } \sup A - \varepsilon < b \le \sup A. $$
