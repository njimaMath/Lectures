集合に関する復習

（＊ ほぼ厳密でない）

§1. 集合と元

集合とは…「もの」の集まり。

元：集合を構成する個々の「もの」

例

  • {猫, 犬, 牛, …}：動物の集合
  • {月, 水, 木, 金, …}：曜日の集合

数学で扱う代表的な集合

  • 自然数：\((1, 2, 3, 4, \dots)\)
    記号：\(\mathbb{N}\)（自然数の集合）

  • 整数：\((0, 1, -1, 2, -2, 3, -3, \dots)\)
    記号：\(\mathbb{Z}\)

  • 有理数：\((0, 1, \frac{1}{3}, 0.25, -\frac{2}{3}, \dots)\)
    記号：\(\mathbb{Q}\)
    → 一般に整数 \(a\), \(b\) を用いて \(\frac{a}{b}\) と表せる

  • 実数：\(\{0, 1, \ldots, \pi, e, \pi e, \ldots\}\)
    記号：\(\mathbb{R}\)

定義

\(a\) が集合 \(A\) の元である時、
\(a \in A\) と書く（\(a\) は \(A\) に含まれる）

そうでない時、\(a \notin A\) と書く。

例

  • \(1 \in \mathbb{N}\)
  • \(2 \in \mathbb{Z}\)
  • \(-2 \notin \mathbb{N}\)

よく使う記法

大学数学では：

  • 「\(a\) を自然数とする」→「\(a \in \mathbb{N}\) とする」と書く
  • 「\(a\) を整数」→「\(a \in \mathbb{Z}\)」と書く
  • ...

関数の収束の記法

関数 $f(x)$ が $x \to a$ のとき実数 $L$ に収束するとき：

$$\lim_{x \to a} f(x) = L $$

と表記する。($x$ が $a$ に近づくと、関数 $f(x)$ が極限値 $L$ にいくらでも近づくことを意味している)

3. 基本的な性質

加法性: $\lim_{x \to a} f(x) = L$, $\lim_{x \to a} g(x) = M$ のとき、$\lim_{x \to a} (f(x) + g(x)) = L + M$

乗法性: $\lim_{x \to a} f(x) = L$, $\lim_{x \to a} g(x) = M$ のとき、$\lim_{x \to a} (f(x) \cdot g(x)) = L \cdot M $

はさみうちの定理: $f(x) \leq g(x) \leq h(x)$ かつ $\lim_{x \to a} f(x) = \lim_{x \to a} h(x) = L$ ならば、$\lim_{x \to a} g(x) = L$

4. \varepsilon-\delta 論法

定義: \begin{definition} 関数 $f(x)$ が $x \to a$ のとき実数 $L$ に収束するとは、任意の $\varepsilon > 0$ に対して、ある $\delta > 0$ が存在して、$0 < |x - a| < \delta$ ならば $|f(x) - L| < \varepsilon$ となることをいう。 \end{definition} 証明の書き方: \epsilon>0を任意に取る.　\delta>0を \delta = [epsilonの関数]ととる。このとき、0 < |x - a| < \delta ならば |f(x) - L| <...< \varepsilon となる。 証明の際は[epsilonの関数]と...をうめる必要がある。
