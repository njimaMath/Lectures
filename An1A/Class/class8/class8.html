数学1A 第8回レジュメ

多変数関数・偏微分・可微分性・接平面

授業のねらい

この回では，一変数関数の微分を二変数関数へ広げる。

Goals
  • 多変数関数の定義域，グラフ，等高線を図で理解する。
  • 偏微分を「一つの方向だけの変化率」として理解する。
  • 偏微分可能性と可微分性の違いを区別する。

2変数関数の見方

$\mathbb R^2$
入力
$(x,y)$
$\xrightarrow{\ f\ }$
$\mathbb R$
出力
$z=f(x,y)$
$\longrightarrow$
曲面・等高線

二変数関数は，平面上の点 $(x,y)$ に数 $z$ を対応させる。

3変数関数の見方

$\mathbb R^3$
入力
$(x,y,z)$
$\xrightarrow{\ f\ }$
$\mathbb R$
出力
$w=f(x,y,z)$
$\longrightarrow$
空間分布・等値面

3変数関数は，空間内の点 $(x,y,z)$ に数 $w$ を対応させる。一定値 $f(x,y,z)=c$ を満たす点の集合は等値面として見る。

1. 多変数関数

2つ以上の変数をもつ関数を多変数関数という。

  • 2変数関数は $z=f(x,y)$ と書く。
  • 3変数関数は $w=f(x,y,z)$ と書く。

工学では，温度，圧力，電位，速度場などが多変数関数として現れる。

たとえば金属板の温度は，位置 $(x,y)$ によって値が変わるので $T(x,y)$ と表せる。

定義域

関数が意味をもつ入力全体を定義域という。二変数関数では，定義域は平面内の領域として表されることが多い。

Check
  • 分母が $0$ にならない範囲を考える。
  • ルートの中が $0$ 以上になる範囲を考える。
  • 対数の中が正になる範囲を考える。

定義域の例

グラフと等高線

2変数関数 $z=f(x,y)$ のグラフは空間内の曲面になる。

また，定数 $c$ に対して

$$f(x,y)=c$$

を満たす点の集合を等高線という。

等高線は，曲面を水平な平面 $z=c$ で切ったときの影を $xy$ 平面に描いたものである。

$z=x^2+y^2$

例題：多変数関数 1

Problem

次の関数の等高線を考えよ。

$f(x,y)=\frac{1}{4} x^2+y^2$

例題：多変数関数 1 解答

Problem

次の関数の等高線を図示せよ。

$f(x,y)=\frac{1}{4} x^2+y^2$

Answer

$f(x,y)=c$ とおくと $\frac{1}{4}x^2+y^2=c$ である。

$c\ge 0$ のとき，原点を中心とする楕円になる。$c<0$ の等高線はない。

例題：多変数関数 2

Problem

次の関数のグラフの形を説明せよ。

$f(x,y)=\frac{1}{4} x+y$

例題：多変数関数 2 解答

Problem

次の関数のグラフの形を説明せよ。

$f(x,y)=\frac{1}{4} x+y$

Answer

グラフは $z=\frac{1}{4} x+y$ で表される平面である。

$x$ 方向に $1$ 増やすと $z$ も $1$ 増え，$y$ 方向に $1$ 増やしても $z$ は $1$ 増える。

例題：多変数関数 3

Problem

次の関数の等高線を考えよ。

$f(x,y)=\frac{1}{4} x-y$

例題：多変数関数 3 解答

Problem

次の関数の等高線を考えよ。

$f(x,y)=\frac{1}{4} x-y$

Answer

$f(x,y)=c$ とおくと $\frac{1}{4}x-y=c$，すなわち $y=\frac{1}{4}x-c$ である。

等高線は傾き $\frac{1}{4}$ の直線になる。

2. 偏微分とは

多変数関数で，ある1つの変数だけを変化させ，他の変数を固定して微分することを偏微分という。

二変数関数 $f(x,y)$

$\dfrac{\partial f}{\partial x}$ は $y$ を定数とみなして $x$ で微分したもの。

$\dfrac{\partial f}{\partial y}$ は $x$ を定数とみなして $y$ で微分したもの。

一方向だけを見る

偏微分係数の定義

点 $(a,b)$ における $x$ に関する偏微分係数は

$$\frac{\partial f}{\partial x}(a,b)=\lim_{h\to 0}\frac{f(a+h,b)-f(a,b)}{h}$$

同様に，$y$ に関する偏微分係数は

$$\frac{\partial f}{\partial y}(a,b)=\lim_{h\to 0}\frac{f(a,b+h)-f(a,b)}{h}$$

記号

偏導関数は次のように書く。

$\frac{\partial f}{\partial x},\quad \frac{\partial f}{\partial y}$

文脈によっては $\frac{\partial f}{\partial x}(x,y)$，$\frac{\partial f}{\partial y}(x,y)$ のように変数を明記する。

幾何学的意味

$\dfrac{\partial f}{\partial x}$ は $x$ 方向の変化率，$\dfrac{\partial f}{\partial y}$ は $y$ 方向の変化率を表す。

曲面 $z=f(x,y)$ を $y=b$ で切ると，一変数関数 $z=f(x,b)$ が得られ，その傾きが $\frac{\partial f}{\partial x}(a,b)$ である。

例題：偏微分 1

Problem

次の関数の $x$ に関する偏微分を求めよ。

$f(x,y)=x^2+3xy+y^2$

例題：偏微分 1 解答

Problem

次の関数の $x$ に関する偏微分を求めよ。

$f(x,y)=x^2+3xy+y^2$

Answer

$y$ を定数とみなして $x$ で微分する。

$\frac{\partial f}{\partial x}(x,y)=2x+3y$

例題：偏微分 2

Problem

次の関数の $y$ に関する偏微分を求めよ。

$f(x,y)=x^2\sin y$

例題：偏微分 2 解答

Problem

次の関数の $y$ に関する偏微分を求めよ。

$f(x,y)=x^2\sin y$

Answer

$x$ を定数とみなして $y$ で微分する。

$\frac{\partial f}{\partial y}(x,y)=x^2\cos y$

例題：偏微分 3

Problem

次の関数の偏導関数 $\frac{\partial f}{\partial x},\ \frac{\partial f}{\partial y}$ を求めよ。

$f(x,y)=e^{xy}$

例題：偏微分 3 解答

Problem

次の関数の偏導関数 $\frac{\partial f}{\partial x},\ \frac{\partial f}{\partial y}$ を求めよ。

$f(x,y)=e^{xy}$

Answer

合成関数の微分を用いる。

$\frac{\partial f}{\partial x}(x,y)=ye^{xy},\quad \frac{\partial f}{\partial y}(x,y)=xe^{xy}$

例題：偏微分 4

Problem

次の関数の偏導関数を求めよ。

$f(x,y,z)=x^2y+yz^3$

例題：偏微分 4 解答

Problem

次の関数の偏導関数を求めよ。

$f(x,y,z)=x^2y+yz^3$

Answer

各変数以外を定数とみなして微分する。

$\frac{\partial f}{\partial x}=2xy,\quad \frac{\partial f}{\partial y}=x^2+z^3,\quad \frac{\partial f}{\partial z}=3yz^2$

3. 偏微分可能性と可微分性

偏微分可能

点 $(a,b)$ において $\frac{\partial f}{\partial x}(a,b)$，$\frac{\partial f}{\partial y}(a,b)$ が存在するとき，その点で偏微分可能という。

注意

偏微分可能であっても，必ずしも関数が"同時に微分"できるとは限らない。

ランダウの記号

定義：$g(x)$ が $x\to0$ のとき $o(x)$ であるとは，次を満たすことである。

$\lim_{x\to0}\frac{g(x)}{x}=0$

つまり，$g(x)$ は $x$ よりも速く $0$ に近づく関数であることを表す。

ランダウの記号の例

Example 1

$x^2=o(x)$：$\frac{x^2}{x}=x\to0$

Example 2

$x\sin x=o(x)$：$\frac{x\sin x}{x}=\sin x\to0$

Not Example

$x$ は $o(x)$ でない：$\frac{x}{x}=1\not\to0$

可微分

関数 $f(x,y)$ が点 $(a,b)$ で可微分であるとは，次を満たすことである。

$f(a+h,b+k)-f(a,b)=\frac{\partial f}{\partial x}(a,b)h+\frac{\partial f}{\partial y}(a,b)k+o(\sqrt{h^2+k^2})$

ここで，$o(\sqrt{h^2+k^2})$ は $(h,k)\to(0,0)$ のとき $\frac{o(\sqrt{h^2+k^2})}{\sqrt{h^2+k^2}}\to0$ となる項を表す。

注意：可微分なら偏微分可能である。たとえば $k=0$ として $h\to0$ とすれば $x$ 方向，同様に $h=0$ として $k\to0$ とすれば $y$ 方向の偏微分係数が得られる。

注意：可微分であるとき，$h,k$ が十分小さければ $f(a+h,b+k)\approx f(a,b)+\frac{\partial f}{\partial x}(a,b)h+\frac{\partial f}{\partial y}(a,b)k$ と近似できる。

例：可微分な関数 1

Example

$f(x,y)=xy$ は任意の点 $(a,b)$ で可微分である。

Reason

$f(a+h,b+k)-f(a,b)=(a+h)(b+k)-ab=bh+ak+hk$

$\frac{\partial f}{\partial x}(a,b)=b$，$\frac{\partial f}{\partial y}(a,b)=a$ なので，一次の部分は $bh+ak$ である。

$r=\sqrt{h^2+k^2}$ とおくと，余り $hk$ は $\frac{|hk|}{r}\le\frac{r^2}{2r}=\frac{r}{2}\to0$ を満たす。

したがって $hk=o(r)$ であり，$f$ は可微分である。

例：可微分な関数 2

Example

$f(x,y)=x^2+y^2$ は任意の点 $(a,b)$ で可微分である。

Reason

$f(a+h,b+k)-f(a,b)=2ah+2bk+h^2+k^2$

$\frac{\partial f}{\partial x}(a,b)=2a$，$\frac{\partial f}{\partial y}(a,b)=2b$ なので，一次の部分は $2ah+2bk$ である。

$r=\sqrt{h^2+k^2}$ とおくと，余り $h^2+k^2$ は $\frac{h^2+k^2}{r}=r\to0$ を満たす。

したがって $h^2+k^2=o(r)$ であり，$f$ は可微分である。

重要な十分条件

$\frac{\partial f}{\partial x}$，$\frac{\partial f}{\partial y}$ が点の近くで存在し，かつ連続ならば，$f$ はその点で可微分である。

$\frac{\partial f}{\partial x},\ \frac{\partial f}{\partial y}$ が近くで連続
$\Longrightarrow$
$f$ は可微分
$\Longrightarrow$
$f$ は偏微分可能

逆向きは一般には成り立たない。

対偶をとると

上の矢印は，対偶をとると次のように使える。

$f$ は偏微分可能でない
$\Longrightarrow$
$f$ は可微分でない

可微分なら偏微分可能である，という命題の対偶である。

偏導関数が連続ならば可微分

点 $(a,b)$ から $(a+h,b+k)$ に動かしたときの変化を，2つの差に分けて見る。

$f(a+h,b+k)-f(a,b)$

$=\{f(a+h,b+k)-f(a+h,b)\}+\{f(a+h,b)-f(a,b)\}$

それぞれを一変数の微分で近似すると，

$\approx \frac{\partial f}{\partial y}(a+h,b)k+\frac{\partial f}{\partial x}(a,b)h$

となる。さらに $(a+h,b)$ が $(a,b)$ に近いので，

$f(a+h,b+k)-f(a,b)\approx \frac{\partial f}{\partial x}(a,b)h+\frac{\partial f}{\partial y}(a,b)k$

偏微分可能だが可微分でない例

次の関数は原点で $\frac{\partial f}{\partial x}(0,0)=0$，$\frac{\partial f}{\partial y}(0,0)=0$ をもつが，可微分ではない。

$$ f(x,y)= \begin{cases} \dfrac{xy}{\sqrt{x^2+y^2}} & ((x,y)\neq(0,0)),\\[0.7em] 0 & ((x,y)=(0,0)). \end{cases} $$

軸方向では値が $0$ になるが，直線 $y=x$ に沿うと $f(x,x)=|x|/\sqrt{2}$ となる。

例題：可微分性 1

Problem

次の関数が点 $(1,2)$ で偏微分可能か調べよ。

$f(x,y)=x^2+xy+y^2$

例題：可微分性 1 解答

Problem

次の関数が点 $(1,2)$ で偏微分可能か調べよ。

$f(x,y)=x^2+xy+y^2$

Answer

多項式なので，すべての点で偏微分可能である。

$\frac{\partial f}{\partial x}=2x+y$，$\frac{\partial f}{\partial y}=x+2y$ より，$\frac{\partial f}{\partial x}(1,2)=4$，$\frac{\partial f}{\partial y}(1,2)=5$ である。

例題：可微分性 2

Problem

次の関数の可微分性を調べよ。

$f(x,y)=x^2y+y^3$

例題：可微分性 2 解答

Problem

次の関数の全微分を求めよ。

$f(x,y)=x^2y+y^3$

Answer

$\frac{\partial f}{\partial x}=2xy$，$\frac{\partial f}{\partial y}=x^2+3y^2$ である。

$df=2xy\,dx+(x^2+3y^2)\,dy$

例題：可微分性 3

Problem

次の関数が可微分であるか考えよ。

$f(x,y)=|x|+|y|$

例題：可微分性 3 解答

Problem

次の関数が可微分であるか考えよ。

$f(x,y)=|x|+|y|$

Answer

$x\ne 0$ かつ $y\ne 0$ の点では可微分である。

$x=0$ または $y=0$ の点では，絶対値の折れ曲がりにより対応する偏微分が存在しないので，可微分ではない。

例題：可微分性 4

Problem

上の $xy/\sqrt{x^2+y^2}$ の例について，偏微分可能性と可微分性の違いを説明せよ。

例題：可微分性 4 解答

Problem

上の $xy/\sqrt{x^2+y^2}$ の例について，偏微分可能性と可微分性の違いを説明せよ。

Answer

原点で $x$ 軸方向と $y$ 軸方向だけを見ると変化率はどちらも $0$ なので，偏微分可能である。

しかし $y=x$ に沿うと $f(x,x)=|x|/\sqrt{2}$ となり，一次近似 $0$ ではこの方向の変化を表せない。したがって原点では可微分でない。

接平面の式

$z=f(x,y)$ が可微分なら，点 $(a,b,f(a,b))$ における接平面は

$$z=f(a,b)+\frac{\partial f}{\partial x}(a,b)(x-a)+\frac{\partial f}{\partial y}(a,b)(y-b)$$

接平面は，曲面を点の近くで一次式に置き換えたものである。これは一変数関数でいう接線の二変数版である。

例題：接平面 1

Problem

次の関数の点 $(1,1)$ における接平面を求めよ。

$z=x^2+y^2$

例題：接平面 1 解答

Problem

次の関数の点 $(1,1)$ における接平面を求めよ。

$z=x^2+y^2$

Answer

$f(1,1)=2$，$\frac{\partial f}{\partial x}(1,1)=2$，$\frac{\partial f}{\partial y}(1,1)=2$ である。

$z=2+2(x-1)+2(y-1)=2x+2y-2$

例題：接平面 2

Problem

次の関数の点 $(0,0)$ における接平面を求めよ。

$z=e^{x+y}+xy$

例題：接平面 2 解答

Problem

次の関数の点 $(0,0)$ における接平面を求めよ。

$z=e^{x+y}+xy$

Answer

$f(0,0)=1$ である。

$\frac{\partial f}{\partial x}=e^{x+y}+y$，$\frac{\partial f}{\partial y}=e^{x+y}+x$ より，$\frac{\partial f}{\partial x}(0,0)=1$，$\frac{\partial f}{\partial y}(0,0)=1$ である。

$z=1+x+y$

例題：接平面 3

Problem

次の関数の原点における接平面を求めよ。

$f(x,y)=x+2y+\dfrac{x^2y^2}{x^2+y^2}\quad ((x,y)\ne(0,0))$

$f(0,0)=0$

例題：接平面 3 解答

Problem

次の関数の原点における可微分性および接平面を求めよ。

$f(x,y)=x+2y+\dfrac{x^2y^2}{x^2+y^2}\quad ((x,y)\ne(0,0))$

$f(0,0)=0$

Answer

$r=\sqrt{h^2+k^2}$ とおくと，

$f(h,k)-f(0,0)=h+2k+\dfrac{h^2k^2}{h^2+k^2}$

余りは $\dfrac{h^2k^2}{h^2+k^2}$ であり，$\dfrac{1}{r}\dfrac{h^2k^2}{h^2+k^2}\le r\to0$ である。

したがって余りは $o(r)$ で，$f$ は原点で可微分である。

$\frac{\partial f}{\partial x}(0,0)=1$，$\frac{\partial f}{\partial y}(0,0)=2$ なので，接平面は

$z=x+2y$
