多変数関数・偏微分・可微分性

二変数関数を図と変化率で見る

授業のねらい

この回では，一変数関数の微分を二変数関数へ広げる。

Goals
  • 多変数関数の定義域，グラフ，等高線を図で理解する。
  • 偏微分を「一つの方向だけの変化率」として理解する。
  • 偏微分可能性と可微分性の違いを区別する。
  • $C^n$ 級関数と混合偏微分の意味を理解する。

2変数関数の見方

$\mathbb R^2$
入力
$(x,y)$
$\xrightarrow{\ f\ }$
$\mathbb R$
出力
$z=f(x,y)$
$\longrightarrow$
曲面・等高線

二変数関数は，平面上の点 $(x,y)$ に数 $z$ を対応させる。

3変数関数の見方

$\mathbb R^3$
入力
$(x,y,z)$
$\xrightarrow{\ f\ }$
$\mathbb R$
出力
$w=f(x,y,z)$
$\longrightarrow$
空間分布・等値面

3変数関数は，空間内の点 $(x,y,z)$ に数 $w$ を対応させる。一定値 $f(x,y,z)=c$ を満たす点の集合は等値面として見る。

1. 多変数関数

2つ以上の変数をもつ関数を多変数関数という。

  • 2変数関数は $z=f(x,y)$ と書く。
  • 3変数関数は $w=f(x,y,z)$ と書く。

工学では，温度，圧力，電位，速度場などが多変数関数として現れる。

たとえば金属板の温度は，位置 $(x,y)$ によって値が変わるので $T(x,y)$ と表せる。

定義域

関数が意味をもつ入力全体を定義域という。二変数関数では，定義域は平面内の領域として表されることが多い。

Check
  • 分母が $0$ にならない範囲を考える。
  • ルートの中が $0$ 以上になる範囲を考える。
  • 対数の中が正になる範囲を考える。

定義域の例

グラフと等高線

2変数関数 $z=f(x,y)$ のグラフは空間内の曲面になる。

また，定数 $c$ に対して

$$f(x,y)=c$$

を満たす点の集合を等高線という。

等高線は，曲面を水平な平面 $z=c$ で切ったときの影を $xy$ 平面に描いたものである。

$z=x^2+y^2$

平面と等温線

例題：多変数関数 1

Problem

次の関数の等高線を考えよ。

$f(x,y)=x^2+y^2$

例題：多変数関数 1 解答

Problem

次の関数の等高線を考えよ。

$f(x,y)=x^2+y^2$

Answer

$f(x,y)=c$ とおくと $x^2+y^2=c$ である。

$c\ge 0$ のとき，原点を中心とする半径 $\sqrt{c}$ の円になる。$c<0$ の等高線はない。

例題：多変数関数 2

Problem

次の関数のグラフの形を説明せよ。

$f(x,y)=x+y$

例題：多変数関数 2 解答

Problem

次の関数のグラフの形を説明せよ。

$f(x,y)=x+y$

Answer

グラフは $z=x+y$ で表される平面である。

$x$ 方向に $1$ 増やすと $z$ も $1$ 増え，$y$ 方向に $1$ 増やしても $z$ は $1$ 増える。

例題：多変数関数 3

Problem

次の関数の等高線を考えよ。

$f(x,y)=x-y$

例題：多変数関数 3 解答

Problem

次の関数の等高線を考えよ。

$f(x,y)=x-y$

Answer

$f(x,y)=c$ とおくと $x-y=c$，すなわち $y=x-c$ である。

等高線は傾き $1$ の平行な直線になる。

例題：多変数関数 4

Problem

次の関数の等高線を考え，中心から離れると値がどう変化するか説明せよ。

$f(x,y)=4-x^2-y^2$

例題：多変数関数 4 解答

Problem

次の関数の等高線を考え，中心から離れると値がどう変化するか説明せよ。

$f(x,y)=4-x^2-y^2$

Answer

$f(x,y)=c$ とおくと $x^2+y^2=4-c$ である。

$c\le 4$ のとき原点中心の円になり，半径は $\sqrt{4-c}$ である。値は原点で最大値 $4$ をとり，中心から離れるほど小さくなる。

例題：多変数関数 5

Problem

工学的な量として，温度分布 $T(x,y)=100-x^2-y^2$ を考える。この関数の意味を説明せよ。

例題：多変数関数 5 解答

Problem

工学的な量として，温度分布 $T(x,y)=100-x^2-y^2$ を考える。この関数の意味を説明せよ。

Answer

点 $(x,y)$ における温度を表す関数である。

原点で温度は最大値 $100$ になり，原点からの距離が大きくなるほど温度は下がる。等温線は原点中心の円である。

2. 偏微分とは

多変数関数で，ある1つの変数だけを変化させ，他の変数を固定して微分することを偏微分という。

二変数関数 $f(x,y)$

$\dfrac{\partial f}{\partial x}$ は $y$ を定数とみなして $x$ で微分したもの。

$\dfrac{\partial f}{\partial y}$ は $x$ を定数とみなして $y$ で微分したもの。

一方向だけを見る

偏微分係数の定義

点 $(a,b)$ における $x$ に関する偏微分係数は

$$f_x(a,b)=\lim_{h\to 0}\frac{f(a+h,b)-f(a,b)}{h}$$

同様に，$y$ に関する偏微分係数は

$$f_y(a,b)=\lim_{h\to 0}\frac{f(a,b+h)-f(a,b)}{h}$$

記号

偏導関数は次のように書く。

$$f_x,\quad f_y,\quad \frac{\partial f}{\partial x},\quad \frac{\partial f}{\partial y}$$

文脈によっては $f_x(x,y)$，$f_y(x,y)$ のように変数を明記する。

幾何学的意味

$\dfrac{\partial f}{\partial x}$ は $x$ 方向の変化率，$\dfrac{\partial f}{\partial y}$ は $y$ 方向の変化率を表す。

曲面 $z=f(x,y)$ を $y=b$ で切ると，一変数関数 $z=f(x,b)$ が得られ，その傾きが $f_x(a,b)$ である。

例題：偏微分 1

Problem

次の関数の $x$ に関する偏微分を求めよ。

$f(x,y)=x^2+3xy+y^2$

例題：偏微分 1 解答

Problem

次の関数の $x$ に関する偏微分を求めよ。

$f(x,y)=x^2+3xy+y^2$

Answer

$y$ を定数とみなして $x$ で微分する。

$f_x(x,y)=2x+3y$

例題：偏微分 2

Problem

次の関数の $y$ に関する偏微分を求めよ。

$f(x,y)=x^2\sin y$

例題：偏微分 2 解答

Problem

次の関数の $y$ に関する偏微分を求めよ。

$f(x,y)=x^2\sin y$

Answer

$x$ を定数とみなして $y$ で微分する。

$f_y(x,y)=x^2\cos y$

例題：偏微分 3

Problem

次の関数の偏導関数 $f_x,\ f_y$ を求めよ。

$f(x,y)=e^{xy}$

例題：偏微分 3 解答

Problem

次の関数の偏導関数 $f_x,\ f_y$ を求めよ。

$f(x,y)=e^{xy}$

Answer

合成関数の微分を用いる。

$f_x(x,y)=ye^{xy},\quad f_y(x,y)=xe^{xy}$

例題：偏微分 4

Problem

次の関数の偏導関数を求めよ。

$f(x,y,z)=x^2y+yz^3$

例題：偏微分 4 解答

Problem

次の関数の偏導関数を求めよ。

$f(x,y,z)=x^2y+yz^3$

Answer

各変数以外を定数とみなして微分する。

$f_x=2xy,\quad f_y=x^2+z^3,\quad f_z=3yz^2$

例題：偏微分 5

Problem

温度分布 $T(x,y)=50+2x-y^2$ について，$\partial T/\partial x$，$\partial T/\partial y$ の意味を説明せよ。

例題：偏微分 5 解答

Problem

温度分布 $T(x,y)=50+2x-y^2$ について，$\partial T/\partial x$，$\partial T/\partial y$ の意味を説明せよ。

Answer

$\partial T/\partial x=2$ なので，$x$ 方向に $1$ 進むと温度は $2$ 上がる。

$\partial T/\partial y=-2y$ なので，$y$ 方向の温度変化は位置 $y$ によって変わる。

3. 偏微分可能性と可微分性

偏微分可能

点 $(a,b)$ において $f_x(a,b)$，$f_y(a,b)$ が存在するとき，その点で偏微分可能という。

注意

偏微分可能であっても，必ずしも関数が滑らかとは限らない。

可微分

関数 $f(x,y)$ が点 $(a,b)$ の近くで一次式によりよく近似できるとき，その点で可微分という。

可微分であれば，

$$f(a+\Delta x,b+\Delta y)$$

は

$$f(a,b)+f_x(a,b)\Delta x+f_y(a,b)\Delta y$$

で近似できる。

全微分と接平面

可微分な関数に対して

$$df=f_xdx+f_ydy$$

を全微分という。小さい変化に対して

$$\Delta f\simeq f_x\Delta x+f_y\Delta y$$

と考えることができる。

接平面の式

$z=f(x,y)$ が可微分なら，点 $(a,b,f(a,b))$ における接平面は

$$z=f(a,b)+f_x(a,b)(x-a)+f_y(a,b)(y-b)$$

接平面は，曲面を点の近くで一次式に置き換えたものである。これは一変数関数でいう接線の二変数版である。

重要な十分条件

$f_x$，$f_y$ が点の近くで存在し，かつ連続ならば，$f$ はその点で可微分である。

$f_x,\ f_y$ が近くで連続
$\Longrightarrow$
$f$ は可微分
$\Longrightarrow$
$f$ は偏微分可能

逆向きは一般には成り立たない。

偏微分可能だが可微分でない例

次の関数は原点で $f_x(0,0)=0$，$f_y(0,0)=0$ をもつが，可微分ではない。

$$ f(x,y)= \begin{cases} \dfrac{xy}{\sqrt{x^2+y^2}} & ((x,y)\neq(0,0)),\\[0.7em] 0 & ((x,y)=(0,0)). \end{cases} $$

軸方向では値が $0$ になるが，直線 $y=x$ に沿うと $f(x,x)=|x|/\sqrt{2}$ となる。

方向による見え方

例題：可微分性 1

Problem

次の関数が点 $(1,2)$ で偏微分可能か調べよ。

$f(x,y)=x^2+xy+y^2$

例題：可微分性 1 解答

Problem

次の関数が点 $(1,2)$ で偏微分可能か調べよ。

$f(x,y)=x^2+xy+y^2$

Answer

多項式なので，すべての点で偏微分可能である。

$f_x=2x+y$，$f_y=x+2y$ より，$f_x(1,2)=4$，$f_y(1,2)=5$ である。

例題：可微分性 2

Problem

次の関数の全微分を求めよ。

$f(x,y)=x^2y+y^3$

例題：可微分性 2 解答

Problem

次の関数の全微分を求めよ。

$f(x,y)=x^2y+y^3$

Answer

$f_x=2xy$，$f_y=x^2+3y^2$ である。

$df=2xy\,dx+(x^2+3y^2)\,dy$

例題：可微分性 3

Problem

次の関数の点 $(1,1)$ における接平面を求めよ。

$z=x^2+y^2$

例題：可微分性 3 解答

Problem

次の関数の点 $(1,1)$ における接平面を求めよ。

$z=x^2+y^2$

Answer

$f(1,1)=2$，$f_x(1,1)=2$，$f_y(1,1)=2$ である。

$z=2+2(x-1)+2(y-1)=2x+2y-2$

例題：可微分性 4

Problem

次の関数が可微分であるか考えよ。

$f(x,y)=|x|+|y|$

例題：可微分性 4 解答

Problem

次の関数が可微分であるか考えよ。

$f(x,y)=|x|+|y|$

Answer

$x\ne 0$ かつ $y\ne 0$ の点では可微分である。

$x=0$ または $y=0$ の点では，絶対値の折れ曲がりにより対応する偏微分が存在しないので，可微分ではない。

例題：可微分性 5

Problem

上の $xy/\sqrt{x^2+y^2}$ の例について，偏微分可能性と可微分性の違いを説明せよ。

例題：可微分性 5 解答

Problem

上の $xy/\sqrt{x^2+y^2}$ の例について，偏微分可能性と可微分性の違いを説明せよ。

Answer

原点で $x$ 軸方向と $y$ 軸方向だけを見ると変化率はどちらも $0$ なので，偏微分可能である。

しかし $y=x$ に沿うと $f(x,x)=|x|/\sqrt{2}$ となり，一次近似 $0$ では全方向の変化を表せない。したがって原点では可微分でない。

4. $C^n$ 級関数

$C^0$ 級：関数が連続である。

$C^1$ 級：1階偏導関数が存在し，それらが連続である。

$C^2$ 級：2階偏導関数が存在し，それらが連続である。

二変数関数では，$C^1$ 級は $f_x$，$f_y$ が存在して連続であることを意味する。

$C^2$ 級と $C^n$ 級

2階偏導関数には

$$f_{xx},\quad f_{xy},\quad f_{yx},\quad f_{yy}$$

がある。

$C^n$ 級

$n$ 階までのすべての偏導関数が存在し，連続であるとき，$C^n$ 級という。

$C^\infty$ 級は，すべての階数の偏導関数が存在し，連続であることを表す。

条件の強さ

$C^\infty$
$\Longrightarrow$
$C^2$
$\Longrightarrow$
$C^1$
$\Longrightarrow$
$C^0$

右へ行くほど条件が弱い。

混合偏微分

条件がよければ，

$$f_{xy}=f_{yx}$$

が成り立つ。特に $f$ が $C^2$ 級ならば，混合偏微分は等しい。

$f$
$\to$
$f_x$
$\to$
$f_{xy}=f_{yx}$
$\leftarrow$
$f_y$

$C^n$ 級の見方

例題：$C^n$ 級 1

Problem

次の関数が原点で $C^1$ 級でないことを調べよ。

$f(x,y)=|x|+y^2$

例題：$C^n$ 級 1 解答

Problem

次の関数が原点で $C^1$ 級でないことを調べよ。

$f(x,y)=|x|+y^2$

Answer

$x$ 方向の偏微分は $f_x(0,0)=\lim_{h\to0}|h|/h$ となり，極限が存在しない。

したがって原点では $C^1$ 級でない。

例題：$C^n$ 級 2

Problem

次の関数の2階偏導関数を求めよ。

$f(x,y)=x^3y+xy^2$

例題：$C^n$ 級 2 解答

Problem

次の関数の2階偏導関数を求めよ。

$f(x,y)=x^3y+xy^2$

Answer

$f_x=3x^2y+y^2$，$f_y=x^3+2xy$ である。

$f_{xx}=6xy,\quad f_{xy}=3x^2+2y,\quad f_{yx}=3x^2+2y,\quad f_{yy}=2x$

例題：$C^n$ 級 3

Problem

次の関数について，$f_{xy}$ と $f_{yx}$ を求めよ。

$f(x,y)=e^{xy}$

例題：$C^n$ 級 3 解答

Problem

次の関数について，$f_{xy}$ と $f_{yx}$ を求めよ。

$f(x,y)=e^{xy}$

Answer

$f_x=ye^{xy}$，$f_y=xe^{xy}$ である。

$f_{xy}=(1+xy)e^{xy},\quad f_{yx}=(1+xy)e^{xy}$

例題：$C^n$ 級 4

Problem

次の関数が原点で $C^2$ 級でないことを調べよ。

$f(x,y)=x|y|$

例題：$C^n$ 級 4 解答

Problem

次の関数が原点で $C^2$ 級でないことを調べよ。

$f(x,y)=x|y|$

Answer

$f_x(x,y)=|y|$ である。

したがって $f_{xy}(0,0)=\lim_{h\to0}|h|/h$ となり，極限が存在しない。よって原点では $C^2$ 級でない。

例題：$C^n$ 級 5

Problem

次の関数が直線 $x=0$ 上で $C^1$ 級でないことを調べよ。

$f(x,y)=|x|+y^2$

例題：$C^n$ 級 5 解答

Problem

次の関数が直線 $x=0$ 上で $C^1$ 級でないことを調べよ。

$f(x,y)=|x|+y^2$

Answer

直線 $x=0$ 上の任意の点 $(0,b)$ で，$f_x(0,b)=\lim_{h\to0}|h|/h$ となる。

この極限は存在しないので，直線 $x=0$ 上では $C^1$ 級でない。

まとめ

  • 多変数関数は，複数の入力変数をもつ関数である。
  • 偏微分は，1つの変数だけを動かして微分する操作である。
  • 偏微分可能であっても，可微分とは限らない。
  • 偏導関数が連続なら，可微分性が保証される。
  • $C^n$ 級とは，$n$ 階までの偏導関数が連続に存在する関数である。
  • $C^2$ 級なら，$f_{xy}=f_{yx}$ が成り立つ。
