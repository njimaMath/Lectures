1. 平行四辺形の面積 \blacksquare 外積 ベクトル $a=(a_1,a_2,a_3)$，$b=(b_1,b_2,b_3)$ に対して，外積 $a\times b$ を次のように定義する． [ a\times b= \begin{vmatrix} \mathbf i & \mathbf j & \mathbf k\ a_1 & a_2 & a_3\ b_1 & b_2 & b_3 \end{vmatrix} =(a_2b_3-a_3b_2,\ a_3b_1-a_1b_3,\ a_1b_2-a_2b_1) ] \blacksquare 平行四辺形の面積 ベクトル $a,b$ が張る平行四辺形の面積 $S$ は [ S=|a\times b| ] で与えられる． 証明 $a,b$ のなす角を $\theta$ とすると [ \cos\theta=\frac{a\cdot b}{|a||b|} ] であり，平行四辺形の面積は [ S=|a||b|\sin\theta ] である．ここで [ \sin\theta=\sqrt{1-\cos^2\theta} =\sqrt{1-\frac{(a\cdot b)^2}{|a|^2|b|^2}} ] だから [ S =|a||b|\sqrt{1-\frac{(a\cdot b)^2}{|a|^2|b|^2}} =\sqrt{|a|^2|b|^2-(a\cdot b)^2}. ] 一方，$a=(a_1,a_2,a_3)$，$b=(b_1,b_2,b_3)$ とおくと， [ a\times b=(a_2b_3-a_3b_2,\ a_3b_1-a_1b_3,\ a_1b_2-a_2b_1) ] なので [ |a\times b|^2 =(a_2b_3-a_3b_2)^2+(a_3b_1-a_1b_3)^2+(a_1b_2-a_2b_1)^2 ] を展開して整理すると [ |a\times b|^2=|a|^2|b|^2-(a\cdot b)^2 ] となる．したがって [ S^2=|a|^2|b|^2-(a\cdot b)^2=|a\times b|^2 ] より $S=|a\times b|$ が従う． 2. 曲面 $x+y=\text{const}$ の直線を描く SVG が考えられる．） ```svg ``` 3. 曲面の面積 上で定義された曲面 $S$ の面積を [ \operatorname{Area}(S) =\iint_D \sqrt{1+\bigl(f_x(x,y)\bigr)^2+\bigl(f_y(x,y)\bigr)^2},dx,dy ] で与える．（ここで $f_x,f_y$ はそれぞれ $x,y$ に関する偏微分．） 演習 1. $f(x,y)=x^2+y^2$，$(x,y)\in[0,1]\times[0,1]$ のとき，曲面の面積を求めよ． 2. $f(x,y)=xy$，$(x,y)\in[0,1]\times[0,1]$ のとき，曲面の面積を求めよ． 3. $f(x,y)=x+y$，$(x,y)\in \{(x,y): x^2+y^2\leq 1\}$ のとき，曲面の面積を求めよ． 4. 解釈 $D=[a,b]\times[c,d]$ の場合を考える．$x$ 軸を $N$ 等分，$y$ 軸を $M$ 等分する． $\Delta x=(b-a)/N,\ \Delta y=(d-c)/M$ とする． $x_i=a+i\Delta x\ (i=0,1,\dots,N)$，$y_j=c+j\Delta y\ (j=0,1,\dots,M)$ とする． $D_{ij}=[x_{i-1},x_i]\times[y_{j-1},y_j]$ とする． 各点 $(x,y)\in D_{ij}$ に対して，$S$ 上の点 $(x,y,f(x,y))$ を対応させる． $D_{ij}$ の 4 頂点を [ A_1=(x_{i-1},y_{j-1}),\ A_2=(x_i,y_{j-1}),\ A_3=(x_i,y_j),\ A_4=(x_{i-1},y_j) ] とし，対応する曲面上の点を [ A_k'=(x_k,y_k,f(x_k,y_k))\quad (k=1,2,3,4) ] と書く（ここでは記号の簡略化のために用いている）． $D_{ij}$ に対応する曲面の部分 [ S_{ij}={(x,y,f(x,y))\mid (x,y)\in D_{ij}} ] は，$A_1',A_2',A_3',A_4'$ を頂点とする平行四辺形で近似できると考えられる． したがって，$S_{ij}$ の面積は，ベクトル $A_2'-A_1'$ と $A_4'-A_1'$ が張る平行四辺形の面積で近似できる． [ A_2'-A_1'=(\Delta x,\ 0,\ f(x_i,y_{j-1})-f(x_{i-1},y_{j-1})) ] [ A_4'-A_1'=(0,\ \Delta y,\ f(x_{i-1},y_j)-f(x_{i-1},y_{j-1})) ] とおくと，$S_{ij}$ の面積 $S_{ij}$ は [ S_{ij}\approx |(A_2'-A_1')\times(A_4'-A_1')| ] で与えられる． ここで [ \delta_1 = f(x_i,y_{j-1})-f(x_{i-1},y_{j-1}),\quad \delta_2 = f(x_{i-1},y_j)-f(x_{i-1},y_{j-1}) ] とおくと [ A_2'-A_1'=(\Delta x,0,\delta_1),\quad A_4'-A_1'=(0,\Delta y,\delta_2) ] であるから [ (A_2'-A_1')\times(A_4'-A_1') =(-\delta_1\Delta y,\ -\Delta x\delta_2,\ \Delta x\Delta y) ] となり， [ |(A_2'-A_1')\times(A_4'-A_1')|^2 =\Delta x^2\Delta y^2\left{1+\left(\frac{\delta_1}{\Delta x}\right)^2 +\left(\frac{\delta_2}{\Delta y}\right)^2\right} ] したがって [ S_{ij}\approx\Delta x,\Delta y \sqrt{1+\alpha_{ij}^2+\beta_{ij}^2} ] ただし [ \alpha_{ij} =\frac{f(x_i,y_{j-1})-f(x_{i-1},y_{j-1})}{\Delta x},\quad \beta_{ij} =\frac{f(x_{i-1},y_j)-f(x_{i-1},y_{j-1})}{\Delta y} ] であり，これらは $f_x,f_y$ の差商になっている． したがって曲面全体の面積 $S$ は [ S\approx \sum_{i=1}^{N}\sum_{j=1}^{M} S_{ij} \approx \sum_{i=1}^{N}\sum_{j=1}^{M} \Delta x,\Delta y, \sqrt{1+\alpha_{ij}^2+\beta_{ij}^2} ] となる．$\Delta x\to 0,\ \Delta y\to 0$（すなわち $N,M\to\infty$）の極限をとると，$\alpha_{ij}\to f_x(x,y)$，$\beta_{ij}\to f_y(x,y)$ となり，この和は [ S=\iint_D \sqrt{1+(f_x(x,y))^2+(f_y(x,y))^2},dx,dy ] という面積の公式に対応するリーマン和になっていることが分かる．
