Section 2. 逆関数

定義: 定義域$A$上で定義された関数が$B$に値取るとき, $f: A\to B$と書く
例:
  1. $f(x)=x^2$は$f:\mathbb{R}\to [0,\infty)$
  2. $f(x)=\sqrt{x}$は$f: [0,\infty)\to [0,\infty)$
定義: 関数$f:A\to B$を考える. 次を満たす関数$g:B\to A$を$f$の逆関数と呼ぶ:
$$\forall x\in A, \quad g\circ f(x)=x$$ $$\forall y\in B, \quad f\circ g(y)=y$$

逆関数の求め方

$f$を関数(わかっている), $g$を$f$の逆関数(わからない)とする. 関数$g$は$x=f(y)$を$y=$[xの式]の形に解くことで求められる.

理由:
  1. $x=f(y)$とする. 両辺に$g$を適用
  2. $g(x) = g\circ f(y)=y$
例:
  1. $f(x)=x^2+2x$の逆関数を求めよ. (定義域は気にしなくて良い)
  2. $f(x) = e^{x^2}$の逆関数を求めよ. (定義域は気にしなくて良い)

Section 3. 逆三角関数

$f(x) = \sin(x)$ (定義域: $[-\pi/2,\pi/2]$)を考える. $g(x)$を$f$の逆関数とするとき, $g$を$\arcsin$, $\sin^{-1}$と書く。$g(x)$の値を主値と呼ぶ.

横軸が$y$、縦軸が$x$のグラフ。$x=1/2$に対応する$y$の値が$\sin^{-1}(1/2) = \pi/6$となる。

注意: $\sin$の定義域をかえると、異なる値がでてくる。
例:
  1. $\sin^{-1}(0)$を求めよ.
  2. $\sin^{-1}(\sqrt{3}/2)$
  3. $\sin^{-1}(-\sqrt{2}/2)$

$f(x)=\cos x$ (定義域: $[0,\pi]$)と$f(x)=\tan x$ (定義域: $(-\pi/2,\pi/2)$)の逆関数をそれぞれ, $\cos^{-1}$, $\tan^{-1}$と書く.

Section 4. 逆三角関数の微分

$x = \sin(\sin^{-1}(x))$より両辺を微分すると:

$$1 = \cos(\sin^{-1}(x)) \cdot (\sin^{-1}(x))'$$

$y\in [-\pi/2,\pi/2]$のとき, $\cos(y)=\sqrt{1-\sin^2(y)}$ より

$$\cos(\sin^{-1}(x))=\sqrt{1-\sin^2(\sin^{-1}(x))}=\sqrt{1-x^2}$$

であるので,

$$(\sin^{-1}(x))' = \frac{1}{\cos(\sin^{-1}(x))} = \frac{1}{\sqrt{1-x^2}}$$
逆三角関数の導関数:
$$(\cos^{-1}(x))' = -\frac{1}{\sqrt{1-x^2}}$$ $$(\tan^{-1}(x))' = \frac{1}{1+x^2}$$

これらを使うと次の積分問題を逆三角関数を用いてかける:

$$\int \frac{1}{\sqrt{1-x^2}} dx = \sin^{-1}(x) + C$$ $$\int \frac{1}{1+x^2} dx = \tan^{-1}(x) + C$$

例題

例:
  1. $\tan^{-1}(\sqrt{3})$の値
  2. $\sin^{-1}(\cos^{-1}(x))$の微分
  3. $\displaystyle\int \sin^{-1}(x) dx = x \sin^{-1}(x) - \int \frac{x}{\sqrt{1-x^2}} dx = x \sin^{-1}(x) + \sqrt{1-x^2} + C$
  4. $\displaystyle\int \frac{x^4}{1+ x^2} dx$
