Section 2. 逆関数

定義: 定義域$A$上で定義された関数が$B$に値取るとき, $f: A\to B$と書く
例:
  1. $f(x)=x^2$は$f:\mathbb{R}\to [0,\infty)$
  2. $f(x)=\sqrt{x}$は$f: [0,\infty)\to [0,\infty)$
定義: 関数$f:A\to B$を考える. 次を満たす関数$g:B\to A$を$f$の逆関数と呼ぶ:
$$\forall x\in A, \quad g\circ f(x)=x$$ $$\forall y\in B, \quad f\circ g(y)=y$$

逆関数の求め方

$f$を既知の関数, $b\in B$とする。 $g$を$f$の逆関数としたとき, $g(a)$を求めたいとする。 次の手順で$g$を求める:

  1. $a=g(b)$とおく。 両辺に$f$を適用すると, $f(a)=f(g(b))=b$となる。
  2. $a$について解く。
例:
  1. $f(x)=2x+1$の逆関数$g$について, $g(1)$を求めよ.
  2. $f(x) = x^2+2x$ (定義域$[-1,\infty)$)の逆関数$g$について, $g(0)$を求めよ.

Section 3. 逆三角関数

$f(x) = \sin(x)$ (定義域: $[-\pi/2,\pi/2]$)を考える. $g(x)$を$f$の逆関数とするとき, $g$を$\arcsin$, ${\rm Sin}^{-1}$と書く。$g(x)$の値を主値と呼ぶ.

主値のポイント:
  • $g$の値域は$[-\pi/2, \pi/2]$に限定し, 同じ$x$でも範囲外の値は採用しない。

横軸が$y$、縦軸が$x$のグラフ。$x=1/2$に対応する$y$の値が${\rm Sin}^{-1}(1/2) = \pi/6$となる。

グラフの読み方

  1. 横軸で求めたい角度$y$を探し, 縦軸に対応する$x=\sin(y)$を確認する。
  2. 逆に, 与えられた$x$から縦方向に進み, 曲線と交わる点の横座標が${\rm Sin}^{-1}(x)$となる。
  3. 主値範囲を外れる箇所では別の枝の値が現れるため, グラフの中央帯を意識する。
注意: $\sin$の定義域をかえると、異なる値がでてくる。
例:
  1. ${\rm Sin}^{-1}(0)$を求めよ.
  2. ${\rm Sin}^{-1}(\sqrt{3}/2)$
  3. ${\rm Sin}^{-1}(-\sqrt{2}/2)$

$f(x)=\cos x$ (定義域: $[0,\pi]$)と$f(x)=\tan x$ (定義域: $(-\pi/2,\pi/2)$)の逆関数をそれぞれ, ${\rm Cos}^{-1}$, ${\rm Tan}^{-1}$と書く.

Section 4. 逆三角関数の微分

導出のステップ

  1. $x = \sin({\rm Sin}^{-1}(x))$を用いて, 合成関数の微分を考える。
  2. 両辺を$x$で微分し, 出てきた$\cos({\rm Sin}^{-1}(x))$を整理する。
  3. $y={\rm Sin}^{-1}(x)$とおけば$y\in [-\pi/2,\pi/2]$で, $\cos(y)=\sqrt{1-\sin^2(y)}$ に注意する。
$$1 = \cos({\rm Sin}^{-1}(x)) \cdot ({\rm Sin}^{-1}(x))'$$
$$\cos({\rm Sin}^{-1}(x))=\sqrt{1-\sin^2({\rm Sin}^{-1}(x))}=\sqrt{1-x^2}$$

したがって

$$({\rm Sin}^{-1}(x))' = \frac{1}{\cos({\rm Sin}^{-1}(x))} = \frac{1}{\sqrt{1-x^2}}$$
見落としがちな点: 平方根は常に非負なので, 分母がゼロになる$x=\pm1$では導関数が定義されない。
逆三角関数の導関数:
$$({\rm Cos}^{-1}(x))' = -\frac{1}{\sqrt{1-x^2}}$$

これらを使うと次の積分問題を逆三角関数を用いてかける:

$$\int \frac{1}{\sqrt{1-x^2}} dx = {\rm Sin}^{-1}(x) + C$$

例題

例:
  1. ${\rm Tan}^{-1}(\sqrt{3})$の値
  2. $({\rm Tan}^{-1}(x))' = \frac{1}{1+x^2}$の微分
  3. $\displaystyle\int {\rm Sin}^{-1}(x) dx = x {\rm Sin}^{-1}(x) - \int \frac{x}{\sqrt{1-x^2}} dx = x {\rm Sin}^{-1}(x) + \sqrt{1-x^2} + C$
  4. $\displaystyle\int \frac{x^4}{1+ x^2} dx$
