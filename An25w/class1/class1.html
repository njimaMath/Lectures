逆関数と逆三角関数の基礎

逆関数の存在条件から逆三角関数の微分・積分公式までを一気通貫で整理し、単射性や置換積分のアイデアを可視化する。

1. 関数の復習

$A, B \subset \mathbb{R}$ を集合とする。$A$ から $B$ への関数 $f$ とは、$A$ の各元 $x$ に対して $B$ のただ一つの元 $y$ を対応させる規則であり、$y = f(x)$ と書く。$A$ を定義域、$B$ を値域、像 $f(A)$ を値の取りうる集合と呼ぶ。

例: $f(x) = x^{2}$ は $\mathbb{R}$ から $[0, \infty)$ への関数であり、$f(\mathbb{R}) = [0, \infty)$ が像となる。

2. 逆関数の定義と単射性

逆関数の定義: 関数 $f: X \to Y$ が単射であるとき、$Y$ 上の関数 $f^{-1}: Y \to X$ を定義でき、任意の $x \in X$, $y \in Y$ に対して
$$f(f^{-1}(y)) = y, \quad f^{-1}(f(x)) = x$$
が成り立つ。この $f^{-1}$ を $f$ の逆関数と呼ぶ。
単調な関数（単調増加または単調減少）は単射であるため、区間上で逆関数が定義される。逆関数の存在を判定する際には、定義域を適切に制限することが重要となる。

3. 逆関数の求め方

逆関数を求める手順は次のとおりである。

  1. $y = f(x)$ とおく。
  2. $y$ を固定したまま $x$ を $y$ の式に解き直す。
  3. $x = g(y)$ と表された式を $f^{-1}(y)$ と読み替える。
  4. 必要に応じて $y$ を $x$ に戻し、定義域と値域を確認する。
例: $f(x) = 2x + 3$ の場合、$y = 2x + 3$ から $x = \dfrac{y - 3}{2}$ を得る。したがって $f^{-1}(y) = \dfrac{y - 3}{2}$、定義域と値域は互いに入れ替わる。

4. 逆関数の微分公式

逆関数の導関数: 関数 $f$ が点 $x_{0}$ の近傍で微分可能で $f'(x_{0}) \neq 0$ とする。このとき $y_{0} = f(x_{0})$ の近傍で逆関数 $f^{-1}$ は微分可能であり、
$$(f^{-1})'(y_{0}) = \dfrac{1}{f'(x_{0})} = \dfrac{1}{f'(f^{-1}(y_{0}))}$$
が成り立つ。
導出: 恒等式 $f(f^{-1}(y)) = y$ を $y$ で微分すると連鎖律より $f'(f^{-1}(y)) \cdot (f^{-1})'(y) = 1$ を得る。$f'(f^{-1}(y)) \neq 0$ の範囲で解けば上の公式になる。

5. 逆三角関数の定義

三角関数は周期的で単射ではないため、適切な区間に制限して逆関数を定義する。標準的な選び方を以下にまとめる。

逆三角関数: 三角関数の単射な部分に制限を与え、その部分の逆関数として $\arcsin, \arccos, \arctan$ を定義する。
逆関数 対応する三角関数 値域 定義域 特徴
$\arcsin x$ $\sin y$ ($y \in [-\tfrac{\pi}{2}, \tfrac{\pi}{2}]$) $[-\tfrac{\pi}{2}, \tfrac{\pi}{2}]$ $[-1, 1]$ 奇関数で単調増加
$\arccos x$ $\cos y$ ($y \in [0, \pi]$) $[0, \pi]$ $[-1, 1]$ 単調減少
$\arctan x$ $\tan y$ ($y \in (-\tfrac{\pi}{2}, \tfrac{\pi}{2})$) $(-\tfrac{\pi}{2}, \tfrac{\pi}{2})$ $\mathbb{R}$ 両端に水平漸近線を持つ

6. 逆三角関数の導関数

導関数の公式:
$$(\arcsin x)' = \dfrac{1}{\sqrt{1 - x^{2}}}, \quad (\arccos x)' = -\dfrac{1}{\sqrt{1 - x^{2}}}, \quad (\arctan x)' = \dfrac{1}{1 + x^{2}}$$
証明のヒント: 恒等式 $\sin(\arcsin x) = x$ を微分すると $\cos(\arcsin x) \cdot (\arcsin x)' = 1$ を得る。単位円の関係 $\cos(\arcsin x) = \sqrt{1 - x^{2}}$ を用いれば導関数が求まる。他の逆三角関数も同様に処理できる。
例: $y = \arctan x$ の導関数 $(\arctan x)' = 1/(1+x^{2})$ は $|x|$ が大きいほど小さくなり、グラフが水平漸近線 $y = \pm \tfrac{\pi}{2}$ に近づく様子を示す。

7. 逆三角関数の積分公式

逆三角関数は置換積分と平方完成を組み合わせることで自然に現れる。以下の基本的な積分公式を押さえておくと応用が広がる。

$$\int \dfrac{1}{\sqrt{1 - x^{2}}} \, dx = \arcsin x + C, \quad \int \dfrac{1}{1 + x^{2}} \, dx = \arctan x + C$$ $$\int \dfrac{1}{1 - x^{2}} \, dx = \tfrac{1}{2}\log\left|\dfrac{1 + x}{1 - x}\right| + C = \operatorname{artanh} x + C$$
例: $\displaystyle \int_{0}^{1/2} \dfrac{1}{\sqrt{1 - x^{2}}} \, dx$ は $\left[\arcsin x\right]_{0}^{1/2} = \arcsin(1/2) = \tfrac{\pi}{6}$ となる。平方根を含む積分が逆三角関数に変換されている点に注目する。
$\int \dfrac{1}{a^{2}+x^{2}} \, dx = \dfrac{1}{a} \arctan\left(\dfrac{x}{a}\right) + C$ や $\int \dfrac{1}{\sqrt{a^{2}-x^{2}}} \, dx = \arcsin\left(\dfrac{x}{a}\right) + C$ など、係数 $a$ の有無に注意してスケール調整を行うと計算がスムーズになる。

8. 応用と演習

逆関数や逆三角関数は極限評価、積分計算、微分方程式の解、確率分布の解析など幅広い分野で現れる。定義から性質を導けるように練習しておこう。

  1. $f(x) = x + e^{x}$ が単調増加であることを示し、逆関数の定義域・値域を求めたうえで $(f^{-1})'(1)$ を計算せよ。
  2. $\arcsin x + \arccos x$ が定数であることを証明し、その値を求めよ。
  3. $g(x) = \arctan(\sqrt{x})$ の導関数を求め、合成関数と逆関数の微分公式の使い分けを練習せよ。
  4. $\displaystyle \int \dfrac{1}{4 + x^{2}} \, dx$ を計算し、逆三角関数の積分公式を活用せよ。
