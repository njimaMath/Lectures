第2回：平面と曲面

この授業では、平面が線形関数のグラフとして見なせることや、より一般的な曲面がレベル集合や断面を通じて理解できることを学ぶ。これらの幾何学的解釈は抽象的な数学的概念を視覚化するのに役立つ。

2次元における直線と曲線

2次元の直線

点 $P=(x_0,y_0)$ と法線ベクトル $\mathbf{n} = (a,b)$ が与えられたとき、$(a,b)$ に直交し、$P$ を通過する直線は、次の条件を満たす点 $Q=(x,y)$ の集合として定義される：

$$(Q-P)\cdot (a,b) = (x-x_0,y-y_0)\cdot(a,b) = 0$$

この直線は、$y = -\frac{a}{b}(x-x_0) + y_0$（ただし $b \neq 0$ と仮定）を満たす点 $(x,y)$ の集合と等価である。

$XY$平面における曲線は、通常、方程式 $y=f(x)$ または陰関数として $F(x,y)=0$ で定義される。

曲線の例：

  • • 放物線: $y=x^2$
  • • 円: $y = \pm\sqrt{1-x^2}$ または $x^2 + y^2 = 1$

3次元空間における平面

3次元空間では、関数 $z=f(x,y)$ のグラフは曲面である。曲面の最も単純な例は、線形方程式で与えられる平面である。

平面の一般方程式

一般的な平面の方程式は：

$$Ax + By + Cz + D = 0$$

ここで $(A,B,C)$ は平面に垂直な法線ベクトルである。

$C \neq 0$ であれば、$z$ について解いて平面を次の形式で表すことができる：

$$z = mx + ny + p$$

ここで、$m = -A/C$ は $x$ 方向の傾き、$n = -B/C$ は $y$ 方向の傾き、$p = -D/C$ は $z$ 切片を表す。

例 2.1

平面 $z = 2x - 3y + 5$ を考えてみる。

これは $x$ 方向に傾き $2$、$y$ 方向に傾き $-3$ の平面である。$z$ 切片は $5$ 。

対応する一般形は $-2x + 3y + z - 5 = 0$ で、法線ベクトルは $(-2, 3, 1)$ 。

3次元空間における曲面

3次元空間における曲面は、関数 $z=f(x,y)$ のグラフである。言い換えれば、曲面は $z=f(x,y)$ を満たす点 $(x,y,z)$ の集合である。

一般的な曲面

  1. 放物面: $z = x^2 + y^2$ [アニメーションを見る]
  2. 円錐: $z = \sqrt{x^2 + y^2}$ [アニメーションを見る]
  3. 鞍点面: $z = y^2 - x^2$ または $y = -x^2 + z^2$ [アニメーションを見る]
図2.1: 3次元空間における一般的な曲面の例：放物面、円錐、鞍点面。

曲面の等高線（レベル曲線）

$f(x, y)$ を2変数の実数値関数とする。$f$ のグラフは、$z = f(x, y)$ で与えられる3次元空間の曲面である。この曲面を2次元表現で研究するために、等高線（レベル曲線とも呼ばれる）をよく使用する。

等高線（レベル曲線）

値 $c$ に対応する $f(x, y)$ の等高線（またはレベル曲線）は、次の条件を満たすすべての点 $(x, y)$ の集合である：

$$f(x, y) = c$$

これは $xy$ 平面上の曲線である。完全な等高線図は、いくつかの $c$ の値に対するこれらの曲線の集まりである。

例えば、$f(x, y) = x^2 + y^2$ で $c = 0, 1, 2$ の等高線は、原点を中心とする半径がそれぞれ $0, 1, \sqrt{2}$ の円である。

例 2.2

問題： 次の関数に対してレベル $f(x, y) = 0$ での等高線を求めなさい：

a) $f(x, y) = \sqrt{1 - x^2 - y^2}$

$f(x, y) = 0$ とすると：

$$\sqrt{1 - x^2 - y^2} = 0$$ $$1 - x^2 - y^2 = 0$$ $$x^2 + y^2 = 1$$

これは原点を中心とする半径1の円。

b) $f(x, y) = 1 - x + 2y$

$f(x, y) = 0$ とすると：

$$1 - x + 2y = 0$$ $$x = 1 + 2y$$

これは傾き2で $x$ 切片が1の直線。

等高線に関する考察

  • 等高線は曲面上の等しい高さ（または値）の点を結ぶ。
  • 等高線は交差しない（ただし、xy平面に投影された場合の鞍点を除く）。
  • 等高線図は地形学、天気図、工学解析で広く使用される。

等高線の具体例：鞍点型関数

関数 $f(x,y)=-x^2+y^2$ の等高線を考察する。等高線は次の方程式で与えられる：

$$f(x,y)=c \quad\Longleftrightarrow\quad y^2 - x^2 = c$$

この等高線は、$c$ の値によって異なる形状を示す：

1. $c>0$ の場合

$$y^2 - x^2 = c$$

これは双曲線で、主軸は $y$ 軸方向に開く。

  • 頂点は $(0,\pm\sqrt{c})$
  • 漸近線は $y=\pm x$

形状は上向きと下向きの2つの枝をもつ。

2. $c=0$ の場合

$$y^2 - x^2 = 0 \;\Longrightarrow\; y=\pm x$$

直線 $y=x$ と $y=-x$ の合併となり、これは退化した双曲線と呼ばれる。

3. $c<0$ の場合

$$y^2 - x^2 = c \;\Longleftrightarrow\; x^2 - y^2 = -c$$

これも双曲線だが、主軸は $x$ 軸方向に開く。

  • 頂点は $(\pm\sqrt{-c},0)$
  • 漸近線はやはり $y=\pm x$

形状は左右に開く2枝の形をもつ。

図2.2: $f(x,y)=-x^2+y^2$ の等高線。$c$ が正から負へ変わるにつれて、開き方が縦方向→退化→横方向へ切り替わる様子が分かる。
