第2節：変数分離法

変数分離形の微分方程式：

$$P(y)\, y' = Q(x) \tag{★}$$

この形式では、$y$に関する関数とその微分がすべて左辺に、$x$に関する関数がすべて右辺にあります。

解法手順：

  1. 両辺を積分して、$$\int P(y)\,dy = \int Q(x)\,dx$$ を計算する。
  2. 可能であれば、\(y\)について解く。

方程式から $\int P(y)\,dy = \int Q(x)\,dx$ の導出：

  1. $a \in \mathbb{R}$ を固定する。
  2. $y = f(x)$ とおくと、 $$ \begin{aligned} \int Q(x)\,dx &= \int_a^x Q(s)\,ds + C \\ &\overset{\text{★}}{=} \int_a^x P(f(s)) f'(s)\,ds + C \\ &\overset{\text{変数変換}}{=} \int_{f(a)}^{f(x)} P(t)\,dt + C \\ &= \int_{f(a)}^y P(t)\,dt = \int P(y)\,dy. \end{aligned} $$

例題と応用

例題1

$$\log y = \int \cos x \, dx - \frac{x^2}{2} + C$$

例題2

$y' = \frac{x}{y}$

例題3

$y' = x (y^2+1),\,y(0)=1$

第3節：定数変化法

一階線形微分方程式：

$y' + P(x)y = Q(x)$

解法手順：

  1. Step 1:$y' + P(x) y=0$の一般解を求める.
  2. Step 1で求めた解の未知定数を$f(x)$に変えて, $y'+P(x) y =Q(x)$が成り立つように$f$を決める.

例題と応用

例題1

$$y' + xy = x$$

解法手順：

  1. $y' + xy=0$を解くと, $y = C e^{-x^2/2}\,$.
  2. $y= f(x)\,C e^{-x^2/2}$とおいて，$y' + xy = x$に代入すると， $$x = y' + xy = f'(x)\,e^{-x^2/2}\,.$$
  3. これを解くと， $$f'(x)= x e^{x^2/2},$$ よって $$f(x)=\int x e^{x^2/2}\,dx = e^{x^2/2}+C\,.$$
  4. 従って一般解は $$y=C\,f(x)\,e^{-x^2/2} =(e^{x^2/2}+C)e^{-x^2/2} =1+C\,e^{-x^2/2}.$$ となる。

例題2

$$y' - y = e^x$$

例題3

$$y' + 3 x^2 y = 3x^3 + 3x^2+ 1.$$
