第10回：$\varepsilon$-$N$論法による極限の証明

1. $\varepsilon$-$N$論法の定義

数列 $(a_n)$ が実数 $a$ に収束するとは、任意の $\varepsilon > 0$ に対して、 ある自然数 $N$ が存在して $$n \ge N \;\Rightarrow\; |a_n - a| < \varepsilon$$ が成り立つことを意味します。このとき、$\lim_{n \to \infty} a_n = a$ と書きます。

2. 極限が既知の場合の証明の書き方

例1：$\displaystyle\lim_{n \to \infty}2a_n = 2a$ の証明

任意の $\varepsilon > 0$ に対して、$\lim a_n = a$ よりある $N$ が存在し、 $$n \ge N \;\Rightarrow\; |a_n - a| < \frac{\varepsilon}{2}$$ となります。よって $$|2a_n - 2a| = 2|a_n - a| < 2 \cdot \frac{\varepsilon}{2} = \varepsilon$$ したがって、$\lim 2a_n = 2a$ が示されました。

例2：$\displaystyle\lim_{n \to \infty}(a_n + 1) = a + 1$ の証明

任意の $\varepsilon > 0$ に対して、$\lim a_n = a$ よりある $N$ が存在し、 $$n \ge N \;\Rightarrow\; |a_n - a| < \varepsilon$$ となります。よって $$|(a_n + 1) - (a + 1)| = |a_n - a| < \varepsilon$$ したがって、$\lim (a_n + 1) = a + 1$ が成り立ちます。

3. 極限が二つある場合の和・積の極限

命題：$\displaystyle\lim a_n = a,\;\lim b_n = b$ のとき、

  • 和：$\displaystyle\lim (a_n + b_n) = a + b$
  • 積：$\displaystyle\lim (a_n b_n) = ab$

和の場合

任意の $\varepsilon > 0$ に対し、$\lim a_n = a$ より $N_1$、$\lim b_n = b$ より $N_2$ が存在して $$n \ge N_1 \Rightarrow |a_n - a| < \tfrac{\varepsilon}{2},\quad n \ge N_2 \Rightarrow |b_n - b| < \tfrac{\varepsilon}{2}.$$ ここで $N = \max(N_1, N_2)$ とすると $$ |(a_n + b_n) - (a + b)| \le |a_n - a| + |b_n - b| < \tfrac{\varepsilon}{2} + \tfrac{\varepsilon}{2} = \varepsilon, $$ したがって $\lim (a_n + b_n) = a + b$ です。

積の場合

収束する数列は有界なので、例えば $|a_n|\le M$ を満たす定数 $M>0$ が取れます。 任意の $\varepsilon>0$ について $\displaystyle\delta = \min\Bigl(1,\frac{\varepsilon}{2(|b|+1)}\Bigr)$ とし、 $n \ge N$ で $|a_n - a| < \delta,\;|b_n - b| < \delta$ とすると

$$ |a_n b_n - ab| \le |a_n - a|\,|b_n| + |a|\,|b_n - b| < \delta(|b|+1) + |a|\delta \le \varepsilon. $$ よって $\lim (a_n b_n) = ab$ が成立します。
