第12回　数列の極限 (ε–N 論法)

1. 極限の定義

数列 $\{a_n\}$ が実数 $a$ に収束するとは，次を満たすことをいう：

$$ \forall \varepsilon > 0\; \exists N \in \mathbb N\;\, \forall n > N : |a_n - a| < \varepsilon. $$

このとき $\displaystyle \lim_{n\to\infty} a_n = a$ と書き，$a_n \to a$ と略記する。

2. ε–N 証明の作法

2.1 1 つの既知極限を利用する場合
（$\displaystyle \lim_{n\to\infty} a_n = a \;\Rightarrow\; \lim_{n\to\infty} b_n = b$ など）

手順

  1. $\lim a_n = a$ を
     $\forall \varepsilon_1>0 \;\exists N_1 \;\forall n>N_1 : |a_n-a|<\varepsilon_1$
    の形で書き下ろす。
  2. 目標の $\varepsilon>0$ を任意に取る。
  3. 適当な関数 $\varepsilon_1 = f(\varepsilon)$ を設定し， 極限の定義から対応する $N_1$ を確保する。
  4. $N := N_1$ あるいは $N := g(\varepsilon,N_1)$ の形で自然数を決める。
  5. $n>N$ ならば連鎖的不等式により $|b_n-b|<\varepsilon$ が示される。

Q. なぜ $\varepsilon$ を $\varepsilon_1$ に置き換えてよいのか？
A. $\varepsilon_1$ も 0 より大きい任意の正数としてとれるため， 定義の「任意の正数」に対応しているから。

2.2 2 つの既知極限を組み合わせる場合
（$\displaystyle \lim a_n = a,\;\lim b_n = b \;\Rightarrow\; \lim c_n = c$ など）

手順

  1. $\lim a_n = a$ を $\varepsilon_1,N_1$、 $\lim b_n = b$ を $\varepsilon_2,N_2$ を使って書き出す。
  2. 目標 $\varepsilon>0$ を任意に取る。
  3. 適切な $\varepsilon_1 = f_1(\varepsilon)$， $\varepsilon_2 = f_2(\varepsilon)$ を決め，それぞれ $N_1,N_2$ を確保する。
  4. $N := \max\{N_1,N_2\}$（またはさらに条件を加味した式）とする。
  5. $n>N$ ならば三角不等式などにより $|c_n-c|<\varepsilon$ を示す。

2.3 証明を理解する 2 つのアプローチ

  1. 慣れで覚える
    • メリット：アイデアが浮かびやすい／書くスピードが速い
    • デメリット：身に付くまで時間がかかる
  2. ルールを体系的に理解する
    • メリット：常に正しい証明が書ける／習得が比較的短期
    • デメリット：最初は機械的で退屈に感じることがある
補題：$A\subset\mathbb R$ とし， $a:=\sup A$ とする。任意の $\varepsilon>0$ について $a-\varepsilon
定理：$a_n$ が単調増加で、集合 $A := \{a_n \mid n \in \mathbb{N}\}$ が上に有界ならば、$a_n \to \sup(A)$ となる。
$a := \sup(A)$ とおく。任意の $\varepsilon > 0$ を取る。補題より $a - \varepsilon < x$ となる $x \in A$ が存在する。$x = a_N$ とすれば、$n > N$ なら $a - \varepsilon < a_N \le a_n \le a$。 よって $|a_n - a| < \varepsilon$ が成り立ち、$a_n \to a$。

3. 基本性質

数列 $a_n \to a$，$b_n \to b$ とする。次が成り立つ。
  1. $a_n + b_n \to a + b$
  2. $a_n b_n \to ab$
  3. $b \neq 0$ ならば $\displaystyle \frac{a_n}{b_n} \to \frac{a}{b}$

(1) 和の極限

　$\varepsilon > 0$ とする。$a_n \to a$ より $\exists N_1$ s.t. $n > N_1 \Rightarrow |a_n - a| < \varepsilon/2$， 同様に $\exists N_2$ s.t. $n > N_2 \Rightarrow |b_n - b| < \varepsilon/2$。 $N := \max\{N_1, N_2\}$ とすれば

$$ |(a_n + b_n) - (a + b)| \le |a_n - a| + |b_n - b| < \varepsilon. $$

(2) 積の極限

　$\varepsilon > 0$ とする。まず $a_n \to a$ から $|a_n| \le |a| + 1$ を保証する $N_1$ を取る （$n > N_1$ で成立）。また (1) と同様に $|a_n - a| < \delta$，$|b_n - b| < \delta$ となる $N_2$ を取る。ここで $$ |a_n b_n - ab| \le |a_n|\,|b_n - b| + |b|\,|a_n - a|. $$ $|a_n| \le |a| + 1$ を用いて $$ |a_n b_n - ab| \le (|a| + 1)\delta + |b|\delta = (|a| + |b| + 1)\,\delta. $$ $\delta = \varepsilon / (|a| + |b| + 1)$ と選べば $|a_n b_n - ab| < \varepsilon$。 $N := \max\{N_1, N_2\}$ で結論。

(3) 商の極限

　$|b| > 0$ なので $\exists c > 0$ s.t. $|b| \ge c$。 $b_n \to b$ だから $|b_n - b| < c/2$ となる $N_1$ を取る。すると $|b_n| \ge c/2$。 以降 (2) と同様に $$ \Bigl|\frac{a_n}{b_n} - \frac{a}{b}\Bigr| = \Bigl|\frac{a_n b - a b_n}{b_n b}\Bigr| \le \frac{|b|\,|a_n - a| + |a|\,|b_n - b|} {|b_n b|} \le \frac{2}{c^2}\bigl(|b|\,|a_n - a| + |a|\,|b_n - b|\bigr). $$ 右辺を $<\varepsilon$ にする $N$ を同様に取ればよい。

4. 部分列

自然数列 $n_1 < n_2 < n_3 < \dots$ に対し $\{a_{n_k}\}$ を $\{a_n\}$ の部分列という。

例 1　$a_n = (-1)^n$，$n_k = 2k - 1$ とすると $b_k := a_{n_k} = -1$ （定数列）。

例 2　同じく $a_n = (-1)^n$，$n_k = 2k$ とすると $b_k := a_{n_k} = 1$。

収束列の任意の部分列は同じ極限に収束する。
$a_n \to a$ とし，部分列 $a_{n_k}$ を取る。 $\varepsilon > 0$ に対し $\exists N$ such that $n > N \Rightarrow |a_n - a| < \varepsilon$。 $n_k > N$ なら同不等式がそのまま成り立つので $a_{n_k} \to a$。■
(収束の必要条件)　もし $a_n \to a$ が成り立たないならば，互いに異なる 2 つの部分列が異なる極限を持つ。
$a_n$ が収束しないと仮定する。すると $$ \exists \varepsilon_0 > 0,\; \forall N,\, \exists m,n > N : |a_m - a_n| \ge \varepsilon_0. $$ $a_n$ を 2 つの開球 $B(a,\varepsilon_0/2)$ とその補集合に分けて取り出すことで 2 つの部分列が得られ，それぞれの極限が異なることが示せる。 ■

例：$(-1)^n$ は収束しない

背理法による。$\{(-1)^n\}$ が $a$ に収束すると仮定する。 $\varepsilon = 1/2$ を取ると $\exists N$ such that $n > N \Rightarrow |(-1)^n - a| < 1/2$。

しかし $n = 2N$ と $n = 2N + 1$ を比較すると

$$ |1 - a| < 1/2, \qquad |-1 - a| < 1/2. $$ 両不等式を足すと $2 < 1$ で矛盾。ゆえに収束しない。 ■

5. Bolzano–Weierstrass (有界列の部分列収束)

有界列 $\{x_n\}$ は収束部分列を持つ。
$\{x_n\}$ が区間 $[m, M]$ に含まれるとする。

峰点 (peak) の集合を定義：

$$ P := \{n \in \mathbb N : x_k \le x_n \; (\forall k > n)\}. $$

場合分け：

  • Case 1 : $P$ が無限
    $P$ の元を昇順に並べた部分列 $n_1 < n_2 < \dots$ では $x_{n_k}$ が単調減少。 有界単調列なので収束する。
  • Case 2 : $P$ が有限
    $n_0 := \max P$ とする。$n > n_0$ なら $n \notin P$ なので $\exists k > n$ with $x_k > x_n$。 帰納的に $$ n_0 < n_1 < n_2 < \dots,\quad x_{n_1} < x_{n_2} < \dots $$ を構成できる。こうして得た部分列は単調増加で有界， ゆえに収束する。
どちらの場合も収束部分列が存在することが分かる。 ■
