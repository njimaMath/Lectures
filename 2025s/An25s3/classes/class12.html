第12回：陰関数定理

1. 3変数関数の偏微分

3変数関数 $F(x, y, z)$ の偏微分は、特定の1つの変数以外を定数とみなして微分する操作である。

定義:

  • $x$ に関する偏導関数: $F_x(x, y, z) = \frac{\partial F}{\partial x} = \lim_{h \to 0} \frac{F(x+h, y, z) - F(x, y, z)}{h}$
  • $y$ に関する偏導関数: $F_y(x, y, z) = \frac{\partial F}{\partial y} = \lim_{h \to 0} \frac{F(x, y+h, z) - F(x, y, z)}{h}$
  • $z$ に関する偏導関数: $F_z(x, y, z) = \frac{\partial F}{\partial z} = \lim_{h \to 0} \frac{F(x, y, z+h) - F(x, y, z)}{h}$

演習問題

問1. $F(x, y, z) = x^2y^3 + z^4$ の偏導関数を求めよ。

$F_x = 2xy^3$
$F_y = 3x^2y^2$
$F_z = 4z^3$

問2. $F(x, y, z) = \sin(xy) + \cos(z)$ の偏導関数を求めよ。

$F_x = y\cos(xy)$
$F_y = x\cos(xy)$
$F_z = -\sin(z)$

問3. $F(x, y, z) = e^{x+2y} \ln(z)$ の偏導関数を求めよ。

$F_x = e^{x+2y} \ln(z)$
$F_y = 2e^{x+2y} \ln(z)$
$F_z = \frac{e^{x+2y}}{z}$

2. 陰関数

方程式 $F(x, y, z) = 0$ によって、3つの変数のうちの1つが他の2つの変数の関数として定められることがある。例えば、$z$ が $x$ と $y$ の関数 $z = \phi(x, y)$ として表される場合、この $\phi$ を 陰関数 (implicit function) と呼ぶ。

これは、方程式 $F(x, y, \phi(x, y)) = 0$ が恒等的に成り立つことを意味する。すべての関数が陽な形 $z = ...$ で簡単に表せるわけではないため、陰関数の考え方は非常に重要である。

演習問題

問1. $F(x, y, z) = x^2 + y^2 + z^2 - 1 = 0$ から $z$ を $x, y$ の陽関数で表せ。($z \ge 0$ の場合)

$z = \sqrt{1 - x^2 - y^2}$

問2. $F(x, y, z) = x + y + z - 5 = 0$ から $z$ を $x, y$ の陽関数で表せ。

$z = 5 - x - y$

問3. $F(x, y, z) = \frac{x^2}{4} + \frac{y^2}{9} - z = 0$ から $z$ を $x, y$ の陽関数で表せ。

$z = \frac{x^2}{4} + \frac{y^2}{9}$

3. 陰関数定理

方程式 $F(x, y, z) = 0$ で定義される陰関数 $z = \phi(x, y)$ の偏導関数を、元の関数 $F$ の偏導関数を使って直接計算することができる。これが 陰関数定理 である。

定理: 点 $(a, b, c)$ の近傍で $F(x, y, z)$ が連続な偏導関数を持ち、$F(a, b, c)=0$ かつ $F_z(a, b, c) \neq 0$ ならば、$F(x, y, z)=0$ は点 $(a, b)$ の近傍で陰関数 $z = \phi(x, y)$ を一意に定める。このとき、$\phi$ の偏導関数は次式で与えられる。

$$\frac{\partial z}{\partial x} = -\frac{F_x(x, y, z)}{F_z(x, y, z)} \quad , \quad \frac{\partial z}{\partial y} = -\frac{F_y(x, y, z)}{F_z(x, y, z)}$$
演習問題

問1. $F(x, y, z) = x^2 + y^2 + z^2 - 1 = 0$ で定まる陰関数 $z=\phi(x,y)$ の $\frac{\partial z}{\partial x}, \frac{\partial z}{\partial y}$ を求めよ。

$F_x = 2x, \quad F_y = 2y, \quad F_z = 2z$
$\frac{\partial z}{\partial x} = -\frac{2x}{2z} = -\frac{x}{z}$
$\frac{\partial z}{\partial y} = -\frac{2y}{2z} = -\frac{y}{z}$

問2. $F(x, y, z) = xyz - 1 = 0$ で定まる陰関数 $z=\phi(x,y)$ の $\frac{\partial z}{\partial x}, \frac{\partial z}{\partial y}$ を求めよ。

$F_x = yz, \quad F_y = xz, \quad F_z = xy$
$\frac{\partial z}{\partial x} = -\frac{yz}{xy} = -\frac{z}{x}$
$\frac{\partial z}{\partial y} = -\frac{xz}{xy} = -\frac{z}{y}$

問3. $F(x, y, z) = x e^z - y \sin(z) + z - 2 = 0$ で定まる陰関数 $z=\phi(x,y)$ の $\frac{\partial z}{\partial x}$ を求めよ。

$F_x = e^z$
$F_z = x e^z - y \cos(z) + 1$
$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z} = -\frac{e^z}{x e^z - y \cos(z) + 1}$

4. 定理の証明

陰関数定理の公式は、合成関数の連鎖律（chain rule）を用いて導出できる。

$F(x, y, z) = 0$ から定まる陰関数 $z = \phi(x, y)$ を考える。このとき、$x$ と $y$ は独立変数であり、$z$ はそれらの従属変数である。$z = \phi(x, y)$ を元の式に代入すると、以下の恒等式が得られる。

$F(x, y, \phi(x, y)) = 0$

この恒等式の両辺を $x$ で偏微分する。左辺は $x, y, z$ の合成関数と見なせるため、連鎖律を適用する。$z$ が $x$ の関数であることに注意が必要である。

$$\frac{\partial F}{\partial x} \frac{\partial x}{\partial x} + \frac{\partial F}{\partial y} \frac{\partial y}{\partial x} + \frac{\partial F}{\partial z} \frac{\partial z}{\partial x} = 0$$

ここで、$\frac{\partial x}{\partial x} = 1$ であり、$x$ と $y$ は独立変数なので $\frac{\partial y}{\partial x} = 0$ である。これを代入すると、

$$F_x \cdot 1 + F_y \cdot 0 + F_z \frac{\partial z}{\partial x} = 0$$ $$F_x + F_z \frac{\partial z}{\partial x} = 0$$

$F_z \neq 0$ の仮定のもとで、$\frac{\partial z}{\partial x}$ について解くと、公式が得られる。

$$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}$$

同様に、恒等式 $F(x, y, \phi(x, y)) = 0$ の両辺を $y$ で偏微分すると ($\frac{\partial y}{\partial y}=1, \frac{\partial x}{\partial y}=0$)、

$$F_y + F_z \frac{\partial z}{\partial y} = 0$$

となり、以下の公式が導かれる。

$$\frac{\partial z}{\partial y} = -\frac{F_y}{F_z}$$

注: この証明は、$z = \phi(x, y)$ が微分可能であることを前提とする。陰関数定理の完全な証明は、その存在と微分可能性を示すものであり、より高度な解析学の範囲となる。
