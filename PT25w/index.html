コースについて

このサイトは「確率論第1同演習」の講義ノートと演習資料をまとめたページです。矢上キャンパス 12-104 教室で火曜3・4限に開講し、ルベーク積分論に基づく確率空間の構成から、期待値の収束定理や分布の極限定理までを厳密な証明とともに学びます。

シラバス
  • 科目名: 確率論第1同演習
  • 副題: 確率空間・確率変数・確率分布
  • 担当: 中島 秀太 / 横山 聡 ・ 単位: 3
  • 曜日・時限: 火曜 3,4時限 / キャンパス: 矢上 / 教室: 12-104
  • 実施形態: 対面（主として対面） / 対象: 理工学部 数理科学科 3年

ルベーク積分論に基づく確率空間を扱い、期待値の収束定理、特性関数と確率分布の関係、分布の極限定理などを厳密に証明しながら理解します。確率論の理論的基礎を強化し、応用的な確率モデルに挑戦するための準備を整えます。

  • 教科書: 熊谷 隆『確率論 (新しい解析学の流れ)』および講義録
  • 参考書: 講義録記載の参考文献を参照
  • 備考: 授業内容は適宜変更する場合があります。質問は随時受け付けます。
