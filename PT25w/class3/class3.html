Lecture 03

期待値（Lebesgue 積分による定義）

Lebesgue 積分の枠組みを用いて期待値を定義し, 可積分性や収束定理による基本性質を確認する。

イントロダクション

確率空間 $(\Omega, \mathcal F, P)$ と実確率変数 $X$ を固定する。期待値は Lebesgue 積分として定義すると, 収束定理など測度論の結果をそのまま活用できる。

1 期待値の定義

定義（単関数 \(\to\) 非負 \(\to\) 一般）

  • 単関数：$X = \sum_{k=1}^n a_k \mathbf{1}_{A_k}$（$a_k \in \mathbb{R}$, $A_k \in \mathcal F$ 互いに素）なら $\displaystyle \mathbb{E}[X] := \sum_{k=1}^n a_k P(A_k)$。
  • 非負可測関数：$0 \le X$ のとき, 単関数の単調増加列 $X_m \uparrow X$ をとり $\displaystyle \mathbb{E}[X] := \lim_{m \to \infty} \mathbb{E}[X_m] \in [0, \infty]$ と定める。
  • 一般の確率変数：$X = X^+ - X^-$（$X^\pm := \max\{\pm X, 0\}$）と分解し, $\mathbb{E}[X^+] + \mathbb{E}[X^-] < \infty$（可積分）なら $\mathbb{E}[X] := \mathbb{E}[X^+] - \mathbb{E}[X^-]$。どちらかが $\infty$ なら期待値は定義しない。
可積分性（$\mathbb{E}[|X|] < \infty$）を仮定することで, 期待値は有限の実数として一意に定まる。非負のみ扱う場合は自動的に定義できる。

2 基本性質

性質

  • 線形性：可積分 $X, Y$ と $\alpha, \beta \in \mathbb{R}$ に対し $\mathbb{E}[\alpha X + \beta Y] = \alpha \mathbb{E}[X] + \beta \mathbb{E}[Y]$。
  • 単調性：$X \le Y$ がほぼ確実に成り立つとき $\mathbb{E}[X] \le \mathbb{E}[Y]$。
  • 単調収束定理（Beppo–Levi）：$0 \le X_n \uparrow X$ なら $\mathbb{E}[X_n] \uparrow \mathbb{E}[X]$。
  • Fatou の補題：$X_n \ge 0$ なら $\mathbb{E}[\liminf X_n] \le \liminf \mathbb{E}[X_n]$。
  • 優収束定理（DCT）：$X_n \to X$ がほぼ確実, $|X_n| \le Y$ かつ $\mathbb{E}[|Y|] < \infty$ なら $\mathbb{E}[X_n] \to \mathbb{E}[X]$。

3 分布を使った表現

定理（LOTUS／法則による計算）

$g \colon \mathbb{R} \to \mathbb{R}$ 可測で $g(X)$ が可積分なら

$$\mathbb{E}[g(X)] = \int_{\mathbb{R}} g(x)\,P_X(dx).$$

ここで $P_X = P \circ X^{-1}$ は $X$ の分布（法則）であり, 期待値は $P_X$ に関する Lebesgue 積分として書ける。

例

  • 離散分布：$P(X = x_k) = p_k$, $\sum_k p_k = 1$, $\sum_k |g(x_k)| p_k < \infty$ なら $\displaystyle \mathbb{E}[g(X)] = \sum_k g(x_k) p_k$。
  • 密度をもつ連続型：$P_X(dx) = f(x)\,dx$, $\int_{\mathbb{R}} |g(x)| f(x)\,dx < \infty$ なら $\displaystyle \mathbb{E}[g(X)] = \int_{\mathbb{R}} g(x) f(x)\,dx$。

4 注意とよくある落とし穴

注意

  • $\mathbb{E}[X]$ の存在には $\mathbb{E}[|X|] < \infty$ が必要（非負なら自動的に満たす）。
  • $\mathbb{E}[X] = \infty$ と書くより, 「期待値は定義されない」と表現するのが厳密。
  • 期待値の存在は分散の存在を保証しないし, その逆も一般には成り立たない。

5 派生量（参考）

定義（分散と共分散）

$\mathbb{E}[|X|^2] < \infty$ のとき, 分散と共分散を次で定める。

$$\mathrm{Var}(X) := \mathbb{E}\bigl[(X - \mathbb{E}[X])^2\bigr], \qquad \mathrm{Cov}(X, Y) := \mathbb{E}[(X - \mathbb{E}[X])(Y - \mathbb{E}[Y])].$$

二乗可積分性（$L^2$ 可積分）を前提にすると, 偏差の二乗が有限となり, 分散や共分散が計算できる。
