1. 歴史のハイライト
   [17世紀] ゲームとギャンブルから生まれた戦略思考
     - パスカルやフェルマーは偶然のゲームで勝利を導くための定量的戦略を追究し、離散確率論の枠組みを築いた。
     - インサイト: 当時のニュートン力学は決定論的であり、確率的な視点とはまだ交わっていなかった。

   [19世紀] 相転移の謎に挑んだ統計物理
     - マクスウェルやボルツマンは氷・水・水蒸気の相転移を説明するため、巨視的現象を微視的粒子の確率モデルとして捉え直した。
     - インサイト: 初期条件や観測時間に不確かさを導入し、決定論的な力学に意図的に確率を重ね合わせる試みが始まった。

   [20世紀] 量子ゆらぎが切り開いた新たな確率論
     - ボルンらによる量子力学は粒子の位置と速度を同時に確定できないことを示し、連続的な確率モデルと公理的確率論へと発展した。
     - インサイト: 波動関数の確率解釈によって、確率論は物理学の基礎概念へと飛躍した。

2. 確率論が担う役割
   - 自然現象の振る舞いは確率的法則に従うことが多く、複雑なシステムを記述するための不可欠な道具になる。
   - ほぼすべての数学分野が確率的な拡張を持ち、学際的研究を支えている。
   - 確率モデルが前提を与え、統計学がデータから妥当性を検証するという双方向の関係が、仮説検定や推論の基盤を形成する。

3. コイントスで学ぶ確率空間
   標本空間の設計
     - 公平なコインを二回投げるとき、標本空間 Ω は { (表, 表), (表, 裏), (裏, 表), (裏, 裏) } となる。
     - 注記: この標本空間を基盤に条件付き確率や期待値の概念を組み立てていく。
     - インサイト: 標本空間と事象を明示すると、確率の議論が曖昧さから解放される。

   有限確率空間の一般形
     - Ω を有限集合とし、各 ω ∈ Ω に非負の重み p(ω) を割り当て総和を 1 に正規化する。
     - 確率測度 P(A) = Σ_{ω∈A} p(ω) と定めると (Ω, 2^Ω, P) が有限確率空間となる。
     - 例: 三回のコイントス、二つのサイコロ、有限状態マルコフ連鎖など。

   確率の公理 (コルモゴロフ)
     1. P(Ω) = 1
     2. 任意の事象 A について 0 ≤ P(A) ≤ 1
     3. 互いに素な可算族 {A_i} に対し P(⋃ A_i) = Σ P(A_i)
     - 注記: 有限集合の枠を超えると、単点集合の確率が 0 でも集合全体に正の確率を割り当てられる。

Ωが可算であるときは、上記のようにすべての集合に対して、確率を定義することができない。-> レポート

   シグマ代数とボレル集合
     - 集合 Ω 上の族 F が Ω を含み、補集合と可算和について閉じていれば F をシグマ代数と呼ぶ。
     - R^d のボレルシグマ代数は開集合族から生成され、連続モデルの標準的な選択となる。
     - 確率空間は (Ω, F, P) の三つ組であり、P は F 上の可算加法的な写像である。

   確率空間の基本性質
     - P(A^c) = 1 - P(A)
     - A ⊂ B ならば P(A) ≤ P(B)
     - P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
     - P(⋃ A_i) ≤ Σ P(A_i) （可算劣加法性）
     - 注記: 包含排除の公式は有限個の事象にも拡張され、交差部分の情報を補正項として扱う。
