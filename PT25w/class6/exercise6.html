第6回 演習: 連続確率空間と分布の性質

class6.html の連続分布を対象に、密度関数や変換の扱いに慣れます。

問題1: 密度関数の正規化

  • 関数 \(f(x) = C (1 + x^2)^{-2}\) を \(x \in \mathbb{R}\) 上で考える。
  • 定数 \(C\) を決定して確率密度関数となるようにせよ。
  • 得られた分布の平均と分散を求めよ。

問題2: 変数変換

  • 確率変数 \(X\) が連続一様分布 \(\mathrm{Unif}(0,1)\) に従うとする。
  • \(Y = -\ln(1 - X)\) と定義し、\(Y\) の密度関数を導出して指数分布に従うことを示せ。

問題3: 連続型の条件付き分布

  • \((X, Y)\) の同時密度が \(f(x, y) = 2\) （ただし \(0 < y < x < 1\) で、それ以外は 0）とする。
  • このとき \(X\) の周辺密度と \(f_{Y \mid X}(y \mid x)\) を求めよ。
