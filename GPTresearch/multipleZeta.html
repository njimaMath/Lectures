0) Glossary and basic notation

  • Multiple zeta values (MZVs).

    For positive integers r and indices n 1 , ‚Ä¶, n r with n 1 ‚â• 2,

A) Core structure of classical MZVs

1) Zagier‚Äôs dimension conjecture (numbers)

Statement. If d w is the coefficient of t w in

\frac{1}{1-t^2-t^3}=1+t^2+t^3+t^4+2t^5+2t^6+3t^7+\cdots,

then d 0 = 1, d 1 = 0, and d w = d w-2 + d w-3 for w ‚â• 3. Zagier conjectured dim ‚Ñö ùì© w = d w for all w.

What is known. Brown proved the full motivic analogue, so the recursion holds inside the motivic Hopf algebra; equality for actual numbers remains open. Annals ‚Üó

2) Worked micro-examples for newcomers

Shuffle‚Äìstuffle identity

Using stuffle (series products) one obtains

\zeta(2)\zeta(3)=\zeta(2,3)+\zeta(3,2)+\zeta(5).

Shuffle (iterated-integral) formulas give a different expression for the same product; subtracting yields a genuine weight-5 relation.

\zeta(2,1)=\zeta(3).

A depth-2 value collapses to depth 1, illustrating the parity phenomenon guiding many reductions.
